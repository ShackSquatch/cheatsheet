Cheatsheet

HTML Element - Basic HTML

Tag Alternative Name Description
<!DOCTYPE> Document Type Defines the document type
<html> HTML Defines an HTML document
<head> Head Contains the document's metadata/information
<title> Title Defines the document's name
<body> Body Contains the document's visible information
<h1>-<h6> Headings Defines an HTML heading
<p> Paragraph Defines a paragraph within a document
<br> Line Break Inserts a single line break in the document
<hr> Horizontal Break Defines a change in the content's theme
<!--...--> Comment Defines a comment within HTML code

HTML Element - Styles / Semantics

Tag Alternative Name Description
<style> Style Defines style information for a document
<div> Divider Defines a section in a document (block-level)
<span> Span Defines a section in a document (inline)
<header> Header Defines a header for a document or section
<footer> Footer Defines a footer for a document or section
<main> Main The main content of a document
<section> Section A section within a document
<article> Article Defines independent, self-contained content of the document
<aside> Aside Defines content indirectly related to surrounding content
<details> Details Defines additional details that a user can choose to view
<dialog> Dialog Defines a dialog box, subwindow, popup, or modal
<summary> Summary Defines a heading for the <details> element visible to the user
<data> Data Adds a machine-readable translation of content
